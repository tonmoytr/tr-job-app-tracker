# 💼 TR Job App Tracker

## 📌 Description

TR Job App Tracker is a clean, user‑friendly **job application management dashboard** that helps users track job postings, application statuses, interview stages, and more — all in one place. It provides an intuitive interface for reviewing job listings and user actions, designed for productivity and simplicity.

🔗 **Live Link:** [http://tr‑job‑app‑tracker.vercel.app/](http://tr‑job‑app‑tracker.vercel.app/)
💻 **Repository:** [https://github.com/tonmoytr/tr‑job‑app‑tracker](https://github.com/tonmoytr/tr‑job‑app‑tracker)

---

## 📖 README

### 🚀 Project Overview

TR Job App Tracker enables users to monitor and manage job applications with an interactive dashboard interface. Users can log in, view lists of job openings, track the status (e.g., applied, interviewing, offer, rejected), and organize applications by priority. The project demonstrates efficient frontend development with modern UI and utilities.

The site appears to be a **static frontend application** hosted on Vercel with real‑time job list interaction — perfect for job seekers who want a quick overview of application workflows.

## 🧑‍💻 Run Locally

Here’s how to get the project running on your local machine:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/tonmoytr/tr-job-app-tracker.git
cd tr-job-app-tracker
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Start Dev Server

```bash
npm run dev
```

---

### 4️⃣ Open in Browser

Open:

```
http://localhost:3000
```
